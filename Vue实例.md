### Vue实例

#### 创建一个Vue实例
```javaScript
    var vm=new Vue({
        //选项（创建你想要的行为）
    })
```
```
所有的Vue组件都是Vue实例，并接受相同的选项对象；
```

#### 数据与方法
```
1、Vue实例被创建时，它向Vue的响应式系统中加入了其data对象中能找到的所有的属性，
当这些属性改变，视图就会产生响应，即匹配更新为新的值；
2、对一些暂未需要的属性，需要设置初始值；
3、Object.freeze()会阻止修改现有的属性；
4、前缀$：区分用户定义的属性；
```
